# AI智能拍照巡检系统

> 店员只需拍照上传，AI帮你审核。释放审核人力，仅需二次确认。

基于人工智能多模态技术的智能巡检解决方案，通过 Aily + Base（多维表格）实现，让零售、餐饮、仓储的安全合规检查变得简单高效。

## ✨ 核心特性

### 🎯 智能识别
- **AI自动识别**：拍照上传后，AI自动识别巡检项并评估合规性
- **无遗漏检查**：确保所有巡检项准确无误地完成
- **实时反馈**：即时给出审核结果和修改建议

### ⏰ 自动化流程
- **周期性任务**：支持每日、每周自动创建巡检任务
- **任务提醒**：每天自动提醒巡检员未完成的任务
- **任务派发**：Webhook自动触发，新任务实时下发到店铺

### 📊 数据可视化
- **管理大盘**：可视化仪表盘，一览所有门店巡检情况
- **风险预警**：快速定位店铺/工厂安全风险和卫生风险
- **完整记录**：所有巡检数据可追溯，方便管理层查阅分析

## 🏗️ 系统架构

### 三大核心流程

#### 1. 店铺巡检流程
```
AI 店铺巡检助手 
  → 意图判断执行技能 
  → 店铺巡检 
  → AI 识别巡检项 
  → AI 巡检图片是否合规 
  → 发出巡检结果
```

#### 2. 每日提醒流程
```
AI 店铺巡检助手 
  → 自动任务触发（定时执行）
  → 每日提醒 
  → 获取当前未完成任务 
  → 获取当前任务店铺 
  → 获取任务店铺巡检员 
  → 批量发出提醒卡片
```

#### 3. 任务派发流程
```
AI 店铺巡检助手 
  → Webhook 任务触发（Base 自动化）
  → 任务派发 
  → 获取新任务列表 
  → 获取新任务店铺 
  → 获取新任务巡检员 
  → 批量发出任务卡片
```

## 🚀 快速开始

### 本地运行

1. **克隆项目**
```bash
git clone https://github.com/Daniel1357/ai_inspection.git
cd ai_inspection
```

2. **启动开发服务器**
```bash
# 使用 Python
python3 -m http.server 3000

# 或使用 Node.js (如果已安装)
npm install
npm run dev
```

3. **打开浏览器**
```
http://localhost:3000/demo.html
```

### 快捷键

- 按数字键 `1-4` 快速切换页面：
  - `1` - 首页
  - `2` - 任务列表
  - `3` - 拍照巡检
  - `4` - 管理大盘

## 📝 功能示例

### 员工巡检
员工只需和助手进行对话，拍照上传就可以完成巡检任务。

### 任务提醒
每天提醒巡检员当前未完成的任务，提升效率和准时性。

### 修改建议
及时反馈修改建议，让安全意识规范意识不迟到。

### 管理者大盘
多维表格配置端仪表盘页面对巡检情况一览无余，快速定位到店铺/工厂安全风险以及卫生风险。

### 周期性巡检
在「任务模版」表中定义周期性巡检任务，可选周期「每天」「每周」。选择后将在「每天」的早上八点为你创建新的天维度周期性任务，「每周」的周一早上八点为你创建新的周维度周期性任务。

### 任务派发
手动创建巡检任务后，相关的巡检员可以接收到巡检任务列表。周期性巡检也能接收到。
